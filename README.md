# E-Commerce Analytics Dashboard
E-commerce sales and customer behavior analysis dashboard using Power BI, Tableau, Excel, and SQL.

# Overview
This project analyzes e-commerce sales data to evaluate business performance, customer purchasing behavior, product trends, payment preferences, and operational efficiency through interactive dashboards and visual analytics.

# Tools Used
* Power BI
* Tableau
* Excel
* SQL

# Dashboards Included
* Sales Dashboard
* Customer Analytics Dashboard

# Key Insights
* Sales and profit performance tracking
* Customer and seller analysis
* Payment method distribution
* Product category performance analysis
* Shipping and customer review trends
* Top-selling product identification
